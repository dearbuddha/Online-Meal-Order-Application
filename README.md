# Online-Meal-Order-Application
An online meal order service implemented with Spring Boot

* IDEA: Intellij IDEA

* Project Management: Maven 3.3.9

* **Backend: Spring Boot 2.1**
	* Database: MySQL + JAP + Mybatis
	* Cashe: Redis
	* Message push: Websocket

* **RESTful**

* **Frontend: Vue**
